### CREATE a new DB
Creating a new database.

```sql
mysql> CREATE DATABASE eubug
```

### DELETE existing DB
Deleting (or dropping) the existing database.

```sql
mysql> DROP DATABASE eubug
```

### SHOW existing DB
List all existing database.

```sql
mysql> SHOW DATABASES;
```

### USE the DB
You need to choose which DB to use.

```sql
mysql> USE eubug;
```
